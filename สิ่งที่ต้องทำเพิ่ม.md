 1. การเตรียมความพร้อมสู่เวอร์ชันจริง (Production Hardening)
    แก้ไขจุดที่เป็นช่องโหว่และอุปสรรคสำหรับการปล่อยแอป (Release Blockers) ตามมาตรฐาน Enterprise:
   * Offline & Security:
       * Bundle Marked.js: ติดตั้งผ่าน npm แทนการโหลดจาก CDN (jsdelivr) เพื่อให้แอปทำงานออฟไลน์ได้และปลอดภัยขึ้น
       * Self-hosted Fonts: ดาวน์โหลดฟอนต์ Inter มาเก็บไว้ในแอปเอง (src/assets/fonts/) และยกเลิกการเรียกใช้
         Google Fonts จากภายนอก
       * Tauri Security: ตั้งค่า withGlobalTauri: false เพื่อป้องกันการเข้าถึง API จากภายนอก และกระชับ Content
         Security Policy (CSP) ให้เข้มงวดขึ้น
   * Code Quality:
       * แก้ไข Cargo Clippy (Rust) ทั้งหมด: รวมถึงการทำ Box สำหรับ Enum ขนาดใหญ่ เพื่อเพิ่มประสิทธิภาพการจัดการ
         Memory
       * ทำความสะอาดโค้ด (Lint/Format) ทั้งฝั่ง Frontend และ Backend

  2. การปรับปรุง UI/UX (Tabbar & Board)
   * Maximize on Start: ตั้งค่าให้แอปเปิดขึ้นมาแบบ เต็มจอ (Maximized) โดยอัตโนมัติ
   * ย้ายปุ่ม Subtasks: ย้ายปุ่ม "Toggle Subtasks" จากเดิมที่อยู่ในหน้า Board ไปไว้ที่ Tab bar (แถวเดียวกับปุ่ม List/Board) เพื่อความสะอาดของ UI
   * Visibility Logic: ปรับให้ปุ่ม Subtasks แสดงผลเฉพาะเมื่อผู้ใช้เลือกดูหน้า Board เท่านั้น
   * Recursive Subtasks: ปรับปรุงให้หน้า Board สามารถแสดงผลงานย่อย (Subtasks) ได้ลึกกี่ชั้นก็ได้
     (เดิมแสดงได้แค่ชั้นเดียว)
    *   ไม่มีปุ่มขยาย และ ย่อ เพิ่มปุ่มขยายและย่อด้วย
